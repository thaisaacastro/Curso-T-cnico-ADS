Docker

  • O que é?

    • Docker é um conjunto de produtos de plataforma como serviço que usam virtualização de nível de sistema operacional para entregar software em pacotes chamados contêineres.



  • O que se trata?

    • Trata-se de uma execução de tarefas otimizada e independente. A ideia dos containers é justamente facilitar esse trabalho recorrente, então são isolados uns dos outros e agrupam seus próprios softwares, bibliotecas e arquivos de configuração
