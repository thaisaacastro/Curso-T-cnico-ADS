Bootstrap

  • O que é?

    • Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript



  • O que se trata?

    • É composto por uma série de códigos que se relacionam entre si para disponibilizar funcionalidades específicas. O Bootstrap é utilizado principalmente por programadores front-end, visto que oferece recursos que simplificam e agilizam a criação de sites.
